# Ranjit Singh Bhui - Portfolio Website

A modern, responsive portfolio website built with Flask (Python) showcasing my experience, skills, and projects as an Automation Specialist.

## Features

- **Modern Light Theme**: Clean, professional design with light blue background
- **Responsive Design**: Works seamlessly on desktop, tablet, and mobile devices
- **Dynamic Content**: Easy to update via `portfolio_data.py`
- **Interactive Elements**: Custom cursor, smooth animations, and hover effects
- **Sections**:
  - Hero section with profile photo
  - About Me
  - Technical Skills
  - Professional Experience (grid layout)
  - Featured Projects
  - Education
  - Contact Information

## Tech Stack

- **Backend**: Flask (Python)
- **Frontend**: HTML5, CSS3, JavaScript
- **Fonts**: Google Fonts (Outfit)
- **Icons**: Font Awesome

## Installation

1. Clone the repository:
```bash
git clone https://github.com/YOUR_USERNAME/portfolio-website.git
cd portfolio-website
```

2. Install dependencies:
```bash
pip install flask
```

3. Run the application:
```bash
python app.py
```

4. Open your browser and navigate to:
```
http://127.0.0.1:5000
```
